# 목적
개인 컴퓨터를 지식 기반 베이스로 활용하여 사용자의 요청에 따라 필요한 데이터를 사용자와 AI가 최소한의 토큰으로 저장 및 가공, 활용을 하기 위함이다.
## 하는 일
1. 사용자의 요청에 맞춰 컴퓨터의 설정 파일을 다루고 최적화 한다. 
2. 사용자의 글을 가공 및 활용한다.
3. 사용자가 제작하는 프로그램들을 체계적으로 정리한다.
## 사용자의 관심사
1. 개인 컴퓨터 커스텀 및 리눅스 환경 활용
2. 글쓰기, 나의 생각 정리 및 활용
3. 고리마케팅이라는 변호사 브랜드 마케팅 운영으로 변호사들의 상황, 처우 등
# 컴퓨터 환경
- 현재 환경은 Thinkpad x1 carbon gen7이고, pop! os COSMIC이며, fcitx5를 쓰고 있다.
- 가상환경은 ~/.venv/bin에 있다.
- 모든 클로드 설정 파일은 ~/.config/claude에 있다. (`~/.claude`는 심링크로 ~/.config/claude를 가리킴) 

# Claude Code 설정
sudo 권한이 필요하지 않은 한, 자동으로 허가하여 진행한다. 

## 언어
모든 작업은 **무조건 한국어로** 진행한다.
- 코드 작성, 주석, 설명 등 모든 커뮤니케이션을 한국어로 진행한다.
- 변수명, 함수명 등 코드에서 한국어 사용이 불가피한 부분은 영어로 작성한다.

## 지침
- 대부분의 설정 파일은 /home/ochan/.config에 존재한다. 설정을 건드려야 할 때 해당 폴더를 먼저 찾아 설정 파일을 수정하며, 프로그램 설정 파일은 각 이름의 폴더로 하여 해당 폴더에 저장토록 한다.
- **로그 저장**이 필요할 때는 `로그_저장` 스킬의 규칙을 따른다. 저장 위치: `~/.config/claude/CLAUDE_LOG/`
- **LLM 위키 관련 작업**(전사, 요약, 개념 정리 등)을 진행할 때는 항상 아래 로컬 폴더 구조의 `LLM_Wiki/03_스키마/` 하위에 위치한 규칙과 워크플로우에 따라 정밀하게 행동한다.
